폼제출전에 해당 함수를 호출하여 유효성체크를 할것.
CKEditor를 사용하는 화면에서는 폼의 유효성체크에 앞서
반드시
CKEDITOR.instances.content.updateElement();
를 실행하여 TEXTAREA와 CKEDITOR를 동기화 할것(붉은색 부분은 화면에서 CKEDITOR로 전환한 항목명)
CKEDITOR에는 아래의 체크요소가 적용되지 않음
연결된 TEXTAREA에 속성을 적용하고,
화면에서 보이지 않아 포커스등의 액션은 불가능하지만  메세지 출력은 가능

*** 중요
서브밋은 submit타입이나 img타입을 가급적 자제하고(유효성체크와 관계없이 서브밋이 되는 경향이 있음)
<input type="button"/>의 onclick이벤트를 이용하여 유효성체크후 직접서브밋할것
화면초기화 및 폼체크설정
폼요소속성 설명 사용방법
required 필수입력 <input type="text" name="somename" required="required"/>
maxlength 최대입력가능길이 <input type="text" name="somename" maxlength="5"/> 영문,한글에 관계없이 문자당 1자로 체크
maxbyte 최대입력바이트 <input type="text" name="somename" maxbyte="20"/>
영문, 한글, 중문등 글자셋에 따라 데이터베이스에 저장될 길이를 체크할 때 사용, 데이터베이스의 컬럼바이트수를 설정
chkType 입력문자패턴체크 <input type="text" name="somename" chkType="num"/> 숫자체크및 숫자이외 입력제한

<input type="text" name="somename" chkType="alpha"/> 영문체크 및 영문이외글자 입력제한

<input type="text" name="somename" chkType="alnum"/> 숫자영문체크 및 숫자영문 이외 입력제한

<input type="text" name="somename" chkType="date"/> 날짜체크(숫자8자리 혹은 yyyy-MM-dd), 숫자,-,.,/이외 입력제한

<input type="text" name="somename" chkType="/^[0-9]{5}$/"/> 정규식에의한 입력값 체크
(정규식설정시 반드시 /로 시작해서 /로 종료할 것, 유효한 정규식이어야 함)
errmsg 유저정의메세지 <input type="text" name="somename" chktype="num" errmsg="숫자이외입력불가"/> 유효성체크시 유저정의 메세지 출력
공통자바스크립트에 기본적인 에러메세지가 있으나, 스크립트파일내에 있어 다국어메세지를 적용하기 어려우므로
소스코드내에서 가능한 메세지를 태그를 이용해 적용할 것
